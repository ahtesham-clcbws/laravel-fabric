# 🌼 DaisyUI (Semantic)

DaisyUI is the most popular component library for Tailwind CSS. It uses semantic classes like `btn-primary`, making it extremely lightweight and Mary UI-inspired.

## 🧵 Forging DaisyUI
```bash
php artisan fabric:component daisyui:{section}
```

## 💎 Smart Components
- `button`
- `alert`
- `badge`
- `toggle`
- `input`
- `select`
- `textarea`
- `modal`
- `stat`
- `progress`

## 🧱 Layouts
- `sidebar-layout`
- `top-nav-layout`
- `bento-layout`
- `web-layout`
