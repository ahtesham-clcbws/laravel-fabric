# 🌊 Meraki UI (Marketing)

Meraki UI is a collection of beautiful, responsive Tailwind CSS components. It is the primary engine for high-conversion marketing pages in the Fabric ecosystem.

## 🧵 Forging Meraki UI
```bash
php artisan fabric:component merakiui:{section}
```

## 🧱 Key Sections
- `hero-section`
- `cta-section`
- `contact-section`
- `pricing-section`
- `testimonial-section`

---
> [!NOTE]
> All Meraki UI components in Fabric are 100% tokenized with `{{ PRIMARY }}` colors.
